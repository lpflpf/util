##日常工作类库总结

### lua 类库
`/lua/throttle` nginx + lua 限流削峰脚本

### PHP 类库
`/php/curl-query` CURL 简单封装  
`/php/image-util` 一个通过Imagick绘制单行或多行文本图像的类。  
`/php/string-util` 一个字符串处理的类，提供字符串截断的方法，支持超链接字符串的截断  
`/php/output-util` 一个输出格式化的类。支持JSON、XML，BJSON，MSGPACK、SERIALIZE等

### python 库
`/python/access-log-util` access_log的查找，筛选，统计
